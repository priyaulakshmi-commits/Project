# LearnTrack

LearnTrack is a console-based **Student & Course Management System**
built using **Core Java**.

The application allows an admin to:
- Manage students
- Manage courses
- Handle enrollments
- Track enrollment status

This project is designed to practice **Java fundamentals** such as:
- Classes and objects
- OOP concepts (encapsulation, inheritance, polymorphism)
- Static vs instance members
- Collections (ArrayList)
- Basic exception handling
- Menu-driven console applications

No database or file storage is used.  
All data is stored **in-memory** using collections.

## How to Compile and Run

### Using Command Line

1. Navigate to the `src` directory:
` cd src`

2. Compile the project:
`javac com/airtribe/learntrack/ui/Main.java`

3. Run the application:
`java com.airtribe.learntrack.ui.Main`
